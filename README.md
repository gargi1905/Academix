📚 Academix – AI-Powered Study Companion

Academix is an AI-powered learning platform designed to make studying more interactive, personalized, and efficient. It enables students to upload their study material in PDF format and automatically transforms it into engaging quizzes, concise study notes, and an intelligent chatbot experience.

Using advanced Large Language Models (LLMs), Academix helps learners assess their understanding, identify weak areas, and revise concepts more effectively. The application also tracks user progress, generates downloadable reports, and provides a modern, user-friendly interface for a seamless learning experience.

Key Features
🔐 Secure User Login and Signup
📄 PDF Upload and Text Extraction
🤖 AI-Based Quiz Generation
⏱️ Timed Quiz System
✅ Automatic Evaluation with Explanations
📊 Progress Tracking Dashboard
📈 Performance Analytics
📑 AI-Generated Study Notes
📥 Downloadable Notes and Result PDFs
💬 AI Chatbot for PDF-based Question Answering
🎨 Modern Purple-Themed User Interface
Technologies Used
Frontend: Streamlit
Backend: Python
AI Model: Groq LLM (Llama 3.3)
PDF Processing: PyMuPDF (fitz)
Data Visualization: Plotly
Report Generation: ReportLab
Environment Management: python-dotenv
Objective

The primary goal of Academix is to simplify the learning process by converting static study materials into an interactive AI-assisted learning environment. It helps students improve comprehension, practice concepts through quizzes, revise efficiently with AI-generated notes, and monitor their academic progress over time.
